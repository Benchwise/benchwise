# Migration Guide

Guide for migrating between Benchwise versions.

## Upgrading to v1.0

### Installation

```bash
pip install --upgrade benchwise
```

### Breaking Changes

None - this is the initial release.

### New Features

- PyTest-like decorator syntax
- Multi-provider support
- Built-in metrics
- Results management
- CLI tools

### Deprecated Features

None

## Future Migrations

Migration guides will be added here as new versions are released.
